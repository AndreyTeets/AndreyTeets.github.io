My projects:

  • DotNetDBTools:

    DotNetDBTools is a set of libraries to define, analyze and deploy (publish migrate) relational databases throughout their evolution using database as code approach (including DBMS-agnostic definition).
    Or in other words - state-based (declarative) database version control tools.

    Github: https://github.com/AndreyTeets/DotNetDBTools
    Nuget: https://nuget.org/packages/DotNetDBTools.Definition
  • FileServer:

    FileServer is a minimalistic https file server with a simple password-only authentication. It provides a simple, secure, universal, cross-patform method for transfering files over network (or just viewing them in text mode in browser).

    Github: https://github.com/AndreyTeets/FileServer
